Configuración Inicial Requerida

Parece que es la primera vez que ejecutas la aplicación.

Para poder continuar, necesitas crear el usuario administrador.

Por favor, detén la aplicación (con Ctrl+C) y ejecuta el siguiente comando en tu terminal:

python setup_admin.py

Una vez que hayas creado el usuario, vuelve a iniciar la aplicación con python app.py y podrás acceder.
